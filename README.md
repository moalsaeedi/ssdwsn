# Scalable Software-Defined Wireless Sensor Networks (SSDWSN)

# INSTALLATION

## Install SSDWSN
```
git clone https://github.com/moalsaeedi/ssdwsn.git
cd ssdwsn
source install.sh install
```
## Run the simulation
```
source install.sh run
```
## Uninstall SSDWSN
```
source install.sh uninstall
```
